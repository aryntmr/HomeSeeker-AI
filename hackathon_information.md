# Hackathon Information

## Context

This repository is for the **Amazon Nova AI Hackathon** (hosted on Devpost). The submission deadline is **March 16, 2026 at 5:00 PM PT** — this is imminent. Winners announced ~April 8, 2026.

## Project Requirements

The core solution **must** use at least one of:
- **Nova 2 Lite** — fast, cost-effective reasoning model for everyday tasks
- **Nova 2 Sonic** — speech-to-speech model for conversational/voice AI
- **Nova Act** — service for building agents that automate UI workflows
- **Nova multimodal embeddings** — state-of-the-art multimodal embedding model

Access models via **Amazon Bedrock** or nova.amazon.com/dev. No AWS credits are available — must use own AWS account.

Allowed frameworks: Strands Agents, LangChain, MCP, A2A, or anything else. Nova must be the core.

## Hackathon Categories (pick one)

- **Agentic AI** — agents using Nova reasoning for complex real-world problems
- **Multimodal Understanding** — apps across text, documents, speech, images, or video
- **UI Automation** — Nova Act agents automating web app workflows
- **Voice AI** — real-time conversational voice using Nova 2 Sonic
- **Freestyle** — anything else

## Judging Criteria

- **Technical Implementation (60%)** — quality, effectiveness, Nova integration, system architecture
- **Enterprise/Community Impact (20%)** — business value or community benefit
- **Creativity/Innovation (20%)** — novelty of approach, innovative use of agents/multimodal

Optimize primarily for technical quality and solid Nova integration.

## What to Submit

1. Text description (brief summary, purpose, how it uses Nova)
2. ~3 minute demo video on YouTube/Vimeo/Youku (include `#AmazonNova`)
3. Public or private GitHub repo (if private, share with testing@devpost.com and Amazon-Nova-hackathon@amazon.com)
4. Bonus: Blog post on builder.aws.com with tag `Amazon-Nova`

## AWS / Infrastructure Notes

- No AWS promotional credits are available anymore
- Must use personal AWS account/credentials
- Keep an eye on AWS usage costs — Nova Bedrock charges apply
